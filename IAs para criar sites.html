Passo a Passo para Criar Seu Site no GitHub Pages

Passo 1: Criar uma Conta no GitHub

Se você ainda não tem uma conta no GitHub, siga esses passos:

  1. Vá para GitHub.

  2. Clique em Sign Up (Cadastro).

  3. Crie sua conta com um nome de usuário, email e senha.

  4. Siga as instruções para confirmar sua conta.

Passo 2: Criar um Novo Repositório

Agora, vamos criar um repositório onde você vai hospedar seu site.

  1. Após fazer login no GitHub, clique no ícone do seu perfil no canto superior direito e selecione Your repositories.

  2. Clique no botão New para criar um novo repositório.

  3. Nomeie o repositório como "nomedosite.github.io" (substitua "nomedosite" pelo nome que você preferir, mas mantenha o formato nomedosite.github.io).

  4. Deixe a opção “Public” marcada para que o repositório seja público e acessível na internet.

  5. Não precisa adicionar um README ou licença, então pode desmarcar essas opções.

  6. Clique em Create repository.

Passo 3: Subir o Código para o Repositório

Agora que você criou o repositório, precisamos adicionar o código HTML e CSS.

  1. Na página do repositório recém-criado, clique na opção "uploading an existing file" (fazer upload de arquivos existentes).

  2. Arraste os arquivos do seu site (HTML, CSS e outros) para a área de upload ou clique em Choose your files para selecionar os arquivos do seu computador.

  3. Após o upload dos arquivos, clique em Commit changes (Confirmar alterações).

Passo 4: Ativar o GitHub Pages

Agora que o código foi enviado para o repositório, precisamos ativar o GitHub Pages para que o seu site seja publicado:

  1. Vá para as Configurações do repositório, clicando na aba Settings (à direita, no topo da página do repositório).

  2. Role até a seção GitHub Pages (vai estar lá embaixo, em "Pages").

  3. Em Source, selecione a opção main (ou master, dependendo de qual branch você tem) e clique em Save.

  4. O GitHub vai gerar o link do seu site. Ele vai aparecer na seção GitHub Pages, algo como:

    • https://nomedosite.github.io

Passo 5: Acessar Seu Site

Agora é só acessar o link que o GitHub criou para você! Seu site estará disponível no endereço:

  • https://nomedosite.github.io

Dicas Extras:

  • Se você precisar atualizar o site, basta adicionar novos arquivos ou editar os arquivos já existentes no repositório e fazer um novo commit.

  • Você pode também adicionar um domínio personalizado (se tiver um) para o seu site, mas, para isso, é necessário um passo extra de configuração.
